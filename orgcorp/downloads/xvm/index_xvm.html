XVM+ .rar

 

 

working with WoT 1.2.0.2

 

FIRST,

if you want to use/test this package please make a copy/backup of your mod folders (MODS, RES_MODS) to a different location at your choice.

after you ARE SURE that you have a backup, delete the original folders (MODS, RES_MODS) and just copy those from .rar into WoT main folder.

                                                 ATTENTION !!! do NOT delete folder [RES], that has a different purpose...

 

Contain:

  • XVM (8.6.6.7 nightly build) - please activate your account on their site and select the WN8 option in settings
  • MoG extended - Spoter github
  • Crew extended* - Spoter
  • Statistics* - Tomonik forum
  • spot indicator
  • others .py files for XVM - Korean Random source

         

XVM have some modifications:

  • by default - auto login, locked gold & free xp, return last crew option, modules (the ones free to remove) on multiple tanks, carousel tanks info, etc. 
  • added a widget to hangar to show stats about selected vehicle (can be minimized) and edited the default one
  • replaced the default prebattle countdown timer & current mission at battle start with a simple countdown
  • moved inbattle missions indicators from the center of screen
  • chance of win (based on players WN8 - not the default XVM)
  • only medium2 tab of players panel is activated:

                      - tank name

                      - yy/zzz as vehicle xTE / number of battles in that vehicle

                      - vehicle icon as class type and premium type (modified battleAtlas.dds file with empty icons)

  • inbattle color reflecting WN8 is modified:

                      - tank name in players panel and star marker: all under 1200 is grey, green until 2000, blue until 2900, after purple

                                     -- more relaxing than rainbow view offered by default --

                      - pressing ALT will show the star color as XVM default color table, name, WN8, country flag, clan, clan icon (if is set)

                      - pressing TAB will show the name & tank colored as XVM default color table

  • manual copy/paste of different .png files is need to show:

                      - specific country flag on players panel [how to]

                      - personalized icon as marker over the tank for specific player or specific clan [how to]

  • A$$Hole or Attention inbattle mark on top of $#!% players and in players panels - photo1, photo2 - how to:

                      - 1. enable the option "Comments and contacts group" on XVM site - example 

                      - 2. in battle or after battle add the naughty player to "Ignore list"

                      - 3. after battle go to your ignore list, image, and click on "Edit data" and rename the player exactly as in photo

                      - 4. save and remove the player from ignore list (the modified data is saved on XVM servers and will be in effect everytime you meet that player in battle)

                              -- there is a zombi face .png available in folders (rename without "_face" as in picture)

                              -- more .png files can be added as long as they are named exactly like you edit the players name in "Edit data" field; tip: use symbols that can't be used in                                                 nickname at account creation 

 

Be sure you did a copy/backup before
