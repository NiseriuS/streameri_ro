Crew voices

 

 

There are players who like older mods for crew voices.

Initial mods were created in a period of FMOD, but with ages WoT moved to WWISE and fsb got dusty at the doors of bnk.

This is a revival for some "oldies but goodies" before the sources get lost in time. Here are a few i already updated to curent version of game.

 

 

Install:

copy .wotmod file to "mods/XYZ" folder, where XYZ is curent version of game.

set "Voice Messages in Battle" to "Standard" (ingame sound tab settings).

Working with: 1.2.0.1 version of WoT (last working version found was for [x.x.xx.x]).
